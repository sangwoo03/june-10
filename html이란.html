1.html이란?
1990년도에 "팀 버너스리"라는 분이 만든 프로그래밍 (마크업)언어이다.

































2.html의 약자
하이퍼텍스트(HyperText): 관련 항목을 연결하기 위해 구성된 텍스트 마크업(Markup): 하드카피나 소프트카피로 출력될 모든 활자에 관한 스타일 가이드 언어(Language): 컴퓨터 시스템이 명령어를 이해하고 해석할 때 사용하는 언어.

































3. 사용하는곳
웹 사이트와 웹 페이지 제작, 웹 애플리케이션 인터페이스 제작, 전자책(ebook)제작등 많은곳에 사용된다.
